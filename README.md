# 🚀 Jelajah Tata Surya Interaktif

Website multimedia hiperaktif yang dibuat menggunakan HTML, CSS, dan JavaScript untuk memperkenalkan tata surya secara interaktif.

## Fitur

✅ Informasi planet interaktif  
✅ Animasi planet berputar  
✅ Audio latar dan efek suara  
✅ Video edukasi tata surya  
✅ Kuis interaktif  
✅ Fakta unik acak setiap halaman dibuka  
✅ Responsive untuk desktop dan mobile

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- YouTube Embed
- HTML Audio

## Cara Menjalankan

1. Download atau clone repository.

```bash
git clone https://github.com/username/tata-surya.git
```

2. Buka folder project.

3. Jalankan file:

```text
index.html
```

menggunakan browser.

---

## Deploy ke GitHub Pages

1. Upload project ke GitHub.

2. Masuk ke:

Settings → Pages

3. Pada bagian:

```
Build and deployment
```

pilih:

```
Deploy from a branch
```

4. Pilih branch:

```
main
```

5. Klik Save.

Website akan tersedia pada alamat:

```text
https://username.github.io/nama-repository/
```
